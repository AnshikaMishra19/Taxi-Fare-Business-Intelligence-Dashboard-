# Taxi Fare Business Intelligence Dashboard 🚕

## Overview
Interactive Power BI dashboard for taxi ride analysis.

## Pages
- 🏠 Homepage - Navigation menu
- 📊 Overall Report - Total rides & revenue
- 🚗 Vehicle Type - Vehicle performance
- 💰 Revenue Report - Revenue trends
- ❌ Cancellation Report - Cancellation insights
- ⭐ Rating Report - Customer ratings
- 📋 Summary Report - Overall summary
